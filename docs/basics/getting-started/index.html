Installation

Installing ian is pretty straightforward. All you have to do is to paste this in your terminal:

# Bash/Zsh
python -c "$(curl -fsSL https://raw.githubusercontent.com/thylong/ian/master/install/install.py)"

# Fish
python -c "(curl -fsSL https://raw.githubusercontent.com/thylong/ian/master/install/install.py)"

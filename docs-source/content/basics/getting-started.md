## Installation

Installing ian is pretty straightforward.
All you have to do is to paste this in your terminal:

```bash
# Bash/Zsh
sudo python -c "$(curl -fsSL https://raw.githubusercontent.com/thylong/ian/master/install/install.py)"

# Fish
sudo python -c "(curl -fsSL https://raw.githubusercontent.com/thylong/ian/master/install/install.py)"
```
